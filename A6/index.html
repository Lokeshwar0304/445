TryIT

and longitude of a particular zip code. http://api.openweathermap.org is also used to fetch

Appointment Service

TryIT

This is RestAPI Service to fetch, create, update and delete test appointments. Json file has been used to save all created appointments

1) GET: api/Appointments
This fetches all appointments from the server

2)CREATE & UPDATE: /home/submitAppointment

User entered data is passed as json object to the api which is entered/ updated in the server appointments json file

3)DELETE: api/Appointments?userid={userid}&apptid={apptid}

userid and apptid are taken is input to find the appointment on server and delete it if it is a valid entry. Note: Webstrar is not allowing delete requests and this will raise exception. Delete works in localhost

No external services were used. Json file is stored in server to save the user entered data
